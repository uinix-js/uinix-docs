# Components Recipes

Common component recipes when using <PackageLink name="uinix-ui" />.

> **Notes:**
- This document provides examples written in [React].  The examples can be similarly implemented with any [hyperscript]-based view library.  For further reading, please refer to the <PackageLink name="uinix-ui" /> package for API documentation.
- This guide contains interactive examples.  Feel free to edit the code!

## Overview

Implementing UI components with <PackageLink name="uinix-ui" /> is easy and usually involves fairly simple and terse code as shown in the examples below.  All components can support variants through appropriate string concatenation of the `variant` prop in relation to how variants are defined in `system.styles` (see [ยง `Button`](#button) for an example).

The examples assume a preset `config` and `theme` detailed in [ยง Appendix](#appendix).  You can adjust the examples appropriately for your configuration.  All examples are interactive and you can edit the code.  The examples outlined in this document are based on the list of [theme-ui components].  Feel free to open an [issue] if you want to suggest a component recipe to be added to the list!

## `Box`

```js live scope=ui
() => {
  // a Box is simply an Element.
  const Box = Element;

  return (
    <Box
      as="p"
      bg="brand.primary"
      color="text.inverse"
      p="m"
      styles={{
        ':hover': {
          backgroundColor: 'background.primary',
          color: 'brand.primary',
        },
      }}>
      A Box is simply a configured Element.
    </Box>
  );
}
```

## `Flex`

```js live scope=ui
() => {
  const Flex = (props) => {
    const styles = [
      {display: 'flex'},
      props.styles,
    ];
    return <Element {...props} styles={styles} />;
  };

  return (
    <Flex
      as="p"
      bg="brand.primary"
      color="text.inverse"
      p="m"
      styles={{
        ':hover': {
          backgroundColor: 'background.primary',
          color: 'brand.primary',
        },
      }}>
      A Flex is simply a Box with "display: flex" set.
    </Flex>
  );
}
```

Note `Flex` can also be expressed with the `Layout` component.

```js live scope=ui
<Layout
  as="p"
  bg="brand.primary"
  color="text.inverse"
  p="m"
  styles={{
    ':hover': {
      backgroundColor: 'background.primary',
      color: 'brand.primary',
    },
  }}>
  The same Flex rendered with Layout.
</Layout>
```

An `Element` appropriately configured shorthand style props can also be used over `Flex`.

```js live scope=ui
<Element
  as="p"
  bg="brand.primary"
  color="text.inverse"
  display="flex"
  p="m"
  styles={{
    ':hover': {
      backgroundColor: 'background.primary',
      color: 'brand.primary',
    },
  }}>
  The same Flex rendered with a configured Element.
</Element>
```

## `Button`

```js live scope=ui
  const Button = (props) => {

    <Layout align="center" spacing="s">

      <Button
        variant="small"

        Boop
      </Button>
      <Button
        hidden
        onClick={() => alert('hidden')}>
        Hidden
      </Button>
    </Layout>
  );
}
```

## `Radio`

```js live scope=ui
() => {
  const Radio = ({
    checked,
    defaultChecked,
    label,
    variant = 'default',
    ...rest
  }) => {
    const isChecked = checked || defaultChecked;

    const styles = {
      '> input ~ .checked': {
        display: 'none',
      },
      '> input ~ .unchecked': {
        display: 'flex',
      },
      '> input:checked ~ .checked': {
        display: 'flex'
      },
      '> input:checked ~ .unchecked': {
        display: 'none'
      },
    };

    const RadioIcon = ({isChecked}) => {
      const status = isChecked ? 'checked' : 'unchecked';
      return (
        <Icon
          className={status}
          color={isChecked ? 'brand.primary' : undefined}
          icon={`form.radio.${status}`}
          size="icon.s"
        />
      );
    }

    return (
      <Layout
        as="label"
        align="center"
        styles={styles}
        spacing="s">
        <Element
          {...rest}
          hidden
          as="input"
          checked={checked}
          defaultChecked={defaultChecked}
          type="radio"
        />
        <RadioIcon isChecked />
        <RadioIcon />
        <div>
          {label}
        </div>
      </Layout>
    );
  };

  return (
    <Layout direction="column" spacing="s">
      <Radio
        name="radio"
        label="Option A"
        value="a"
      />
      <Radio
        defaultChecked
        name="radio"
        label="Option B"
        value="b"
      />
    </Layout>
  );
}
```
