# Components

Common component recipes using <PackageLink name="uinix-ui" />.

> **Notes:**
- This document provides examples written in [React].  The examples can be similarly implemented with any [hyperscript]-based view library.  For further reading, please refer to the <PackageLink name="uinix-ui" /> package for API documentation.

## Overview

Implementing UI components with <PackageLink name="uinix-ui" /> is easy and usually involves fairly simple and terse code.  There are many ways to implementing components, which are highlighted in some examples below.  All components can support variants through appropriate string concatenation of the `variant` prop in relation to how variants are defined in `system.styles` (see [ยง `Button`](#button) for an example).

All examples are interactive, so please feel free to edit the code.  The examples outlined in this document are based on the list of [theme-ui components].  Feel free to open an [issue] if you want to suggest a component to be added to the list!

The examples assume a preset `config` (enabling shorthand style props) and `theme`, so make sure to adjust these appropriately in your implementation.

## `Box`

```js live scope=ui
() => {
  const Box = (props) => {
    return <Element {...props} />
  };

  return (
    <Box
      as="p"
      bg="brand.primary"
      color="white"
      p="m"
      styles={{
        ':hover': {
          background: 'white',
          color: 'brand.primary',
        },
      }}>
      A Box is simply a configured Element.
    </Box>
  );
}
```

Note that the `Element` component can be used as a `Box`.

```js live scope=ui
<Element
  as="p"
  bg="brand.primary"
  color="white"
  p="m"
  styles={{
    ':hover': {
      background: 'white',
      color: 'brand.primary',
    },
  }}>
  The same Box rendered with Element.
</Element>
```

## `Flex`

```js live scope=ui
() => {
  const Flex = (props) => {
    const styles = [
      {display: 'flex'},
      props.styles,
    ];

    return (
      <Element
        {...props}
        styles={styles}
      />
    );
  };

  return (
    <Flex
      as="p"
      bg="brand.primary"
      color="white"
      p="m"
      styles={{
        ':hover': {
          background: 'white',
          color: 'brand.primary',
        },
      }}>
      A Flex is simply a Box with "display: flex" set.
    </Flex>
  );
}
```

Note that the `Layout` component can be used `Flex`.

```js live scope=ui
<Layout
  as="p"
  bg="brand.primary"
  color="white"
  p="m"
  styles={{
    ':hover': {
      background: 'white',
      color: 'brand.primary',
    },
  }}>
  The same Flex rendered with Layout.
</Layout>
```

Or even an `Element` with appropriately configured shorthand style props can be a `Flex`.

```js live scope=ui
<Element
  as="p"
  bg="brand.primary"
  color="white"
  display="flex"
  p="m"
  styles={{
    ':hover': {
      background: 'white',
      color: 'brand.primary',
    },
  }}>
  The same Flex rendered with Layout.
</Element>
```

## `Button`

```js live scope=ui
() => {
  const Button = ({variant, ...rest}) => {
    const variant = `button.${variant}`;

    return (
      <Element
        {...rest}
        as="button"
        variant={variant}
      />
    );
  };

  return (
    <Layout align="center" spacing="s">
      <Button onClick={() => alert('beep')}>
        Beep
      </Button>
      <Button
        variant="small"
        onClick={() => alert('boop')}>
        Boop
      </Button>
      <Button
        hidden
        onClick={() => alert('hidden')}>
        Hidden
      </Button>
    </Layout>
  );
}
```

## `Select`

```js live scope=ui
() => {
  const Select = ({
    options,
    variant = 'default',
    ...rest
  }) => {
    return (
      <Element
        {...rest}
        as="select"
        variant={`form.select.${variant}`}>
        {options.map(({label, value}) => (
          <option key={value} value={value}>
            {label}
          </option>

      </Element>
    );

  return (
    <Select
      options={[
        {label: 'Option A', value: 'a'},
        {label: 'Option B', value: 'b'},
        {label: 'Option C', value: 'c'},
      ]}
      w="100%"
    />
  );
}
```

## `Radio`

```js live scope=ui
() => {
  const Radio = ({
    checked,
    defaultChecked,
    label,
    variant = 'default',
    ...rest
  }) => {
    const isChecked = checked || defaultChecked;

    const styles = {
      '> input ~ .checked': {
        display: 'none',
      },
      '> input ~ .unchecked': {
        display: 'flex',
      },
      '> input:checked ~ .checked': {
        display: 'flex'
      },
      '> input:checked ~ .unchecked': {
        display: 'none'
      },
    };

    const RadioIcon = ({isChecked}) => {
      const status = isChecked ? 'checked' : 'unchecked';
      return (
        <Icon
          className={status}
          color={isChecked ? 'brand.primary' : undefined}
          icon={`form.radio.${status}`}
          size="icon.s"
        />
      );
    }

    return (
      <Layout
        as="label"
        align="center"
        styles={styles}
        spacing="s">
        <Element
          {...rest}
          hidden
          as="input"
          checked={checked}
          defaultChecked={defaultChecked}
          type="radio"
        />
        <RadioIcon isChecked />
        <RadioIcon />
        <Element variant="form.input.label">
          {label}
        </Element>
      </Layout>
    );
  };

  return (
    <Layout direction="column" spacing="s">
      <Radio
        name="radio"
        label="Option A"
        value="a"
      />
      <Radio
        defaultChecked
        name="radio"
        label="Option B"
        value="b"
      />
    </Layout>
  );
}
```
